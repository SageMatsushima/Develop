Development

Context

During my sophomore fall semester at Brown, my UI/UX class assigned us a project that focused on React and its useful applications. As a result, the app focused on components and state properties. The app I created focused on the applications React has on online shopping. It is a shop where users can buy Pokemon and add each item to the cart. Its designed to be item efficient and helps the user quickly navigate the website to find what they desire.

The link to the React App

Research

Some usability principles that I considered was:

  • User Control: The user has full freedom of the website and their shopping experience.
  • Flexibility and Efficiency of Use: Filters and sorting helps the user find what they are looking for.
  • Error Prevention: Remove buttons and add buttons help the user if they made a mistake or forgot to add an item.

Using these principles, my React website was based on the user's experience and the goal for the shopping design. The items were clear and the buttons helped prevent any confusion with the interface. It makes it a clear design for users that help them focus on their goal and filters out what they don't want for a more streamlined experience.

Takeaways

After learning React and JS firsthand, I feel more comfortable with the program and its applications. I liked how simple it could be to make a component and repeat products on a website. It saves time for developers and makes it easier to code.
